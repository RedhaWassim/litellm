# EdenAI Platform
EdenAI is a platform that aggregates and provides access to multiple Large Language Models (LLMs) under a single account. With EdenAI, users can interact with different LLMs from different providers using a unified API.

more informations : https://edenai.co/

## EdenAI supports multiple types of requests, allowing users to leverage the full capabilities of modern LLMs:

Normal Calls: Standard text-based interactions with the models.
Multimodal Calls: Interactions that involve both text and other media, such as images.
Tool Calls: Function calling capabilities where models can interact with external tools.
